# MVP計画

## MVP定義

### スコープ

| 項目 | MVP（Phase 1） | Phase 2 |
|------|---------------|---------|
| エリア | 東京都全域（23区 + 多摩地域、約800駅） | 首都圏（神奈川・千葉・埼玉） |
| レイヤー | Safety, Hazard, Vibe（3/7層） | + Sound, Community, Change, Sensory |
| 課金 | なし（全公開） | フリーミアム（月N駅まで無料） |
| UGC | テキスト投稿のみ | 写真・動画、モデレーション、NLP分析 |

### 主要ページ

1. **トップページ**: 東京都マップ（治安エリアカラーマップ） + 駅/エリア検索バー + エリアから探す
2. **駅ページ**（約659ページ、SSG自動生成）: Safety/Hazardセクション + 周辺エリア一覧 + UGC口コミ
3. **エリアページ**（約5,000ページ、SSG+ISR）: Safety/Vibeセクション + UGC口コミ（丁目単位）
4. **市区町村ページ**: 配下エリア一覧（偏差値順）
5. **比較ページ**: 2〜3エリアを横並びで比較

### 技術アーキテクチャ

SSG（静的サイト生成）中心のアーキテクチャ。治安・災害データは月次〜年次更新のため、ビルド時に静的ページを生成。UGCのみクライアントサイドでSupabaseからリアルタイム取得。

```
[データパイプライン(Python)] → [Supabase(PostgreSQL+PostGIS)]
                                         ↓
                               [next build (SSG)]
                                         ↓
                              [Vercel (静的ページ配信)]
                                         ↓
                         [ブラウザ: UGCのみCSRでSupabaseから取得]
```
